# ES2015 Chat

This a a simple ES6 Chat written in Flux with React.

## Installation
 - Install and configure MongoDB
 - Clone the Repository
 - npm install
 - gulp serve


## Available Tasks

Start the Frontend and Backend:
```
gulp serve
```

Run the Tests:
```
gulp test
```

or

```
gulp watch-tests
```

Generate Documentation:
```
gulp doc
```
